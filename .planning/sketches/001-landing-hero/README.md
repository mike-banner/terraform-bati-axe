# Sketch 001: Landing Hero

## Design Question
Quel style de Hero (en-tête de page) donne l'impact le plus premium et convertit le mieux pour un SaaS BTP ?

## How to View
Ouvrir dans le navigateur : `.planning/sketches/001-landing-hero/index.html`

## Variants
- **A: Loud Minimal** — Typographie massive "Exaggerated Minimalism", focus sur le texte et l'espace négatif, background très clair, bouton d'action orange fluo (safety orange).
- **B: Dark Glass** — Ambiance Dark Mode, flou gaussien derrière les textes pour effet "Glow", mode ultra technologique / luxe, parfait pour montrer l'aspect "moteur V4".
- **C: Bento Grid Hero** — Disposition asymétrique type Apple, information très dense mais visuellement digeste, donne immédiatement le contexte et des metrics clés.

## What to Look For
- Regarde quelle "vibe" correspond le mieux au message de BÂTI-AXE.
- Est-ce que le contraste "Gris Industriel / Orange" fonctionne mieux sur fond clair (A/C) ou fond sombre (B) ?
- L'arrangement visuel : Préfères-tu le minimalisme direct (A) ou l'information structurée (C) ?
