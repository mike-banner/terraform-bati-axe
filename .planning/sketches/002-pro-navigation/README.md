# Sketch 002: Navigation Espace Pro

## Design Question
Comment rendre l'espace pro plus "bureau" sur desktop (sans la navbar grand public) et plus "app native" sur mobile (avec une bottom bar type VTC) ?

## How to View
Ouvrir ce fichier dans votre navigateur :
`file:///home/mike/projects/bati-axe/.planning/sketches/002-pro-navigation/index.html`

## Variants
- **A: Sidebar (Gauche) + Bottom Bar Mobile** — Sur Desktop, une barre latérale complète type SaaS.
- **B: Top App Bar (Haut) + Bottom Bar Mobile** — Sur Desktop, une barre en haut très épurée et aérée (Top Navigation).

## What to Look For
1. **Sur Mobile (réduisez la fenêtre)** : Observez la Bottom Bar fixe en bas (type VTC).
2. **Sur Desktop** : Comparez la Variante A (plus "Logiciel métier/Admin") et la Variante B (plus "App grand public/Aéré"). Laquelle correspond le mieux à l'ambiance recherchée ?
