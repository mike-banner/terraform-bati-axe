# PIVOT STRATÉGIQUE : BÂTI-AXE V2 (B2B Prescripteurs)

## 1. La Scission : V1 (Fondations) vs V2 (L'Axe B2B)

### V1 : Le Socle (Ce qui est fait, validé et robuste)
- **Cible initiale** : Particulier (B2C) cherchant un artisan -> Artisan (B2B).
- **Acquis Techniques (La Forteresse)** :
  - Pipeline d'onboarding pro strict (SIRET, Kbis, Décennale).
  - Vérification automatisée API État (Sécurité et conformité légale).
  - Verrou de leads (Floutage) et monétisation Stripe automatisée.
  - Refonte UI (Charte gris/orange industrielle, Bento Grid, propre et B2B).
- **Limites du modèle V1** : L'acquisition B2C est une guerre d'usure coûteuse (SEO/Ads face aux géants). Les particuliers génèrent beaucoup de demandes "curieuses" ou de bas budget.

### V2 : Le Réseau Privé (Le Pivot B2B2C)
- **Nouvelle Cible VIP (Apporteurs d'affaires)** : Agences immobilières, Architectes, Syndics de copropriété, Diagnostiqueurs immobiliers, Assureurs, Experts en sinistres.
- **Le Concept** : Les apporteurs d'affaires (Prescripteurs) "injectent" des chantiers massifs sur la plateforme pour le compte de leurs clients, car notre outil leur simplifie la vie. Le tunnel B2B est un "Fast-Track" (zéro questionnaire, dépôt de plans/rapports directs, SLA de chiffrage sous 48h).
- **L'Avantage Injuste (Moat)** : Un agent immo génère 5 à 10 gros chantiers par an. S'il adopte BÂTI-AXE comme *son* outil de chiffrage, l'acquisition de leads devient gratuite, récurrente et de très haute qualité.

## 2. Adaptation des Nouvelles Phases pour la V2

### Phase 5.5 : Portfolio Pro (Le Dossier d'Appel d'Offres)
- **Vision V1** : Une jolie galerie de photos pour un particulier.
- **Vision V2 (Pivot)** : Un véritable *Dossier de Qualification* pour les pros de l'immo.
  - *Mécanique* : Le profil public (mobile first, sans menu distractif) sert de "CV certifié". L'agent immobilier peut envoyer le lien du profil BÂTI-AXE à son acheteur en disant : *"Cet artisan a les bonnes assurances (vérifiées) et voici ses réalisations dans ce budget"*.

### Phase 5.6 : Calculateur de Prix (L'Outil d'Aide à la Vente)
- **Vision V1** : Un simulateur grand public pour capter un e-mail.
- **Vision V2 (Pivot)** : La "Calculatrice de Poche" du Diagnostiqueur ou de l'Agent.
  - *Mécanique* : Un agent visite un bien avec un DPE G (passoire thermique) qui effraie l'acheteur. Il dégaine BÂTI-AXE sur sa tablette, sélectionne les travaux, sort une estimation en temps réel, et pousse la mise en relation avec nos pros. Le Calculateur lève l'objection de vente de l'agent.

### Phase 5.7 : Pipeline CRM & Tracking Translucide
- **Vision V2** : Le prescripteur veut savoir ce que devient son client.
  - *Mécanique* : Quand l'artisan fait avancer la jauge de progression (ex: "Contacté" -> "Devis envoyé" -> "Chantier Signé"), le Prescripteur est notifié. Ce suivi de bout en bout crée une confiance absolue : le Syndic ou l'Architecte sait qu'il ne recommande pas "dans le vide".

## 3. L'Impact UI/UX (Ce qu'on doit designer)

1. **La Landing Page Splitée (Le Carrefour)** :
   - L'accueil ne dit plus "Trouvez un artisan". Il dit "BÂTI-AXE, le réseau certifié".
   - Bouton 1 : **"Espace Professionnel du Bâtiment"** (Recevoir des chantiers).
   - Bouton 2 : **"Espace Décideur Immobilier"** (Agences, Syndics, Architectes).
   - *(Le flux B2C classique reste accessible mais devient secondaire).*

2. **Le Compte "Prescripteur" (Nouveau Rôle BDD)** :
   - Un accès gratuit pour l'apporteur d'affaires. Son tableau de bord ne montre pas des "Leads", mais "Mes dossiers clients" avec le fameux pipeline de suivi (Phase 5.7).
