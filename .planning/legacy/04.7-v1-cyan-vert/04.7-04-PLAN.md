  <action>
Refactor dashboard pages to apply unified card styling and responsive grid layout.

**Card Base Styling (shared for both pages):**
- Background: #ECFEFF (light cyan, --color-background)
- Border-radius: 12px (--radius-lg)
- Padding: 24px (--space-lg)
- Box-shadow: 0 4px 6px rgba(0,0,0,0.1) (--shadow-md)
- Transition: all 200ms ease
- Cursor: pointer (clickable cards)
- Hover state: shadow-lg (0 10px 15px rgba(0,0,0,0.1)) + translateY(-2px) — NOT scale transform
- Focus-within: outline 2px solid #0891B2, outline-offset 0

**Lead Card Grid Layout (index.vue, lines ~80+):**

1. **Grid Container:**
   - Mobile: `grid grid-cols-1` (1 column)
   - Tablet (768px+): `md:grid-cols-2` (2 columns)
   - Desktop (1024px+): `lg:grid-cols-3` (3 columns)
   - Gap: 24px (--space-lg) both x and y
   - Max-width: 1400px, margin: auto, padding: responsive clamp

2. **Individual Card Structure:**
   ```
   <div class="card">
     <div class="flex justify-between items-start">
       <span class="badge bg-cta text-white">Status</span>
       <span class="text-xs text-gray-500">Category • Date</span>
     </div>
     <h3 class="font-heading text-lg font-600 text-text mt-4">Lead Title</h3>
     <p class="text-sm text-gray-700 mt-3 line-clamp-2">Description</p>
     <div class="flex gap-2 mt-6">
       <button class="btn-secondary">View Details</button>
       <button class="btn-secondary">Contact</button>
     </div>
   </div>
   ```

3. **Card Inner Sections:**
   - Badge (status): `bg-cta (#22C55E) text-white px-2 py-1 rounded text-xs font-500`
   - Metadata (category + date): `text-gray-500 text-xs`
   - Title (h3): `Figtree 600, 18px, dark cyan #164E63`
   - Description (p): `Noto Sans 400, 14px, dark gray #374151, line-clamp-2 (prevent overfow)`
   - Buttons: secondary style (cyan outline, 12px 24px padding)

4. **Lead Age Indicator (if LeadAge component exists):**
   - Keep component styling but ensure colors match MASTER.md
   - Green (<24h), amber (3-7d), red (≥7d)

5. **Filters + Sort Controls (top of index.vue):**
   - Container: bg-background light cyan, padding --space-lg, rounded --radius-md, shadow-sm
   - Label: Figtree 500, 14px
   - Select/checkbox inputs: use MASTER.md input styles (from 04.7-03)
   - Button (apply filter): green primary, hover opacity-90

6. **Pagination (if exists, bottom of index.vue):**
   - Container: margin-top 48px (--space-2xl)
   - Page buttons: secondary style, hover bg-background
   - Current page: primary style or border highlight

**Lead Detail Page Layout ([id].vue):**

1. **Header Section:**
   - Background: white or light cyan
   - Title: Figtree 700, 32px-48px clamp
   - Status badge: same green badge style
   - Back link: cyan text with arrow, hover underline

2. **Sections Container:**
   - Use card style for each section
   - Margin-bottom: 24px (--space-lg)
   - Section heading: Figtree 600, 20px

3. **Lead Information Sections:**
   - Project Description: card with title + paragraphs
   - Client Info (if visible): card with masked name, project budget, timeline
   - Qualification Badges: separate card with checkmarks for budget/description/phone
   - Actions: green buttons for claim/contact/qualify

4. **Responsive Detail Layout:**
   - Mobile: single column, full width sections
   - Tablet (768px+): two-column layout for non-blocking info (e.g., budget + timeline side-by-side)
   - Desktop (1024px+): full layout with sidebar + main content

5. **Typography in Detail:**
   - All headings: Figtree (600 for section headers, 700 for page title)
   - Body: Noto Sans 400, 16px line-height 1.6
   - Labels: Noto Sans 500, 14px, gray

**Banners & Alerts (index.vue, if exists):**
- Success banner (upgrade completed): bg-green-50 border-left border-green-500, text-green-900
- Paywall banner (free leads used): bg-amber-50 border-left border-amber-500, text-amber-900
- Free leads remaining: card with progress bar, text-center

**Colors Audit:**
- Status badge (open): #22C55E (green)
- Status badge (claimed/contacted): #0891B2 (cyan) or gray (#9CA3AF)
- Card background: #ECFEFF (light cyan)
- Card text: #164E63 (dark cyan) for titles, #374151 (dark gray) for body, #6B7280 (medium gray) for metadata
- Button primary (green): #22C55E
- Button secondary (cyan outline): border #0891B2 text #0891B2

**Avoid Anti-Patterns:**
- ❌ No emoji status indicators (use text badges)
- ❌ No missing cursor:pointer on clickable cards
- ❌ No layout-shifting hovers (use shadow + translate, not scale)
- ❌ No low contrast text on cards
- ❌ No instant state changes (all 200-300ms transitions)

**Accessibility:**
- Card headings are proper h3 tags (semantic)
- Buttons inside cards are keyboard accessible (tab order)
- Colors are not the only indicator (use text + color for status)
- Link text is descriptive ("View Details" not "Click Here")
