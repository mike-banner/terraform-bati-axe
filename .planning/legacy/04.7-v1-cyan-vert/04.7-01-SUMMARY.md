# Phase 04.7 Plan 01: CSS Foundation & Design Tokens Summary

**Figtree + Noto Sans Google Fonts integration with MASTER.md color palette and complete design token system**

## What Was Built

Task 1: **Complete rewrite of `app/assets/css/tailwind.css`** with the following changes:

### Fonts
- ✅ Removed `@font-face` declarations for self-hosted Clash Display + Geist Variable
- ✅ Added Google Fonts import: `@import url('https://fonts.googleapis.com/css2?family=Figtree:wght@300;400;500;600;700&family=Noto+Sans:wght@300;400;500;700&display=swap')`
- ✅ Updated `@theme` tokens: `--font-sans: 'Noto Sans'`, `--font-heading: 'Figtree'`

### Colors (MASTER.md Hex Palette)
- ✅ `--color-primary: #0891B2` (cyan — primary links/secondary CTAs)
- ✅ `--color-secondary: #22D3EE` (cyan lighter — hover states)
- ✅ `--color-cta: #22C55E` (green — primary buttons)
- ✅ `--color-background: #ECFEFF` (light cyan page background)
- ✅ `--color-text: #164E63` (dark cyan text)
- ✅ `--color-border: #E2E8F0` (gray borders)
- ✅ `--color-border-error: #DC2626` (red error state)
- ✅ `--color-border-disabled: #F1F5F9` (light gray disabled state)
- ✅ `--color-ring: #0891B220` (cyan with 13% opacity for focus rings)

**Removed:** All OKLCH Brique & Béton variables (--brand, --brand-hover, --brand-foreground, --brand-ink, --surface, --surface-sable, --border-strong, etc.)

### Spacing System
- ✅ `--space-xs: 0.25rem` (4px — tight gaps)
- ✅ `--space-sm: 0.5rem` (8px — icon gaps, inline spacing)
- ✅ `--space-md: 1rem` (16px — standard padding)
- ✅ `--space-lg: 1.5rem` (24px — section padding)
- ✅ `--space-xl: 2rem` (32px — large gaps)
- ✅ `--space-2xl: 3rem` (48px — section margins)
- ✅ `--space-3xl: 4rem` (64px — hero padding)

### Shadow System
- ✅ `--shadow-sm: 0 1px 2px rgba(0, 0, 0, 0.05)` (subtle lift)
- ✅ `--shadow-md: 0 4px 6px rgba(0, 0, 0, 0.1)` (cards, buttons)
- ✅ `--shadow-lg: 0 10px 15px rgba(0, 0, 0, 0.1)` (modals, dropdowns)
- ✅ `--shadow-xl: 0 20px 25px rgba(0, 0, 0, 0.15)` (hero images, featured cards)

### Radius System
- ✅ `--radius-sm: 6px`
- ✅ `--radius-md: 8px` (buttons per MASTER.md)
- ✅ `--radius-lg: 12px` (cards per MASTER.md)

### Component Base Styles (`@layer components`)

## Threat Surface Scan

**Google Fonts Import (via HTTPS):**
- ✅ Using HTTPS URL from fonts.googleapis.com (secure by default)
- ✅ `display=swap` parameter ensures fallback font renders immediately (FOUT instead of FOIT)
- ✅ No custom fonts serve from untrusted sources

**CSS Custom Properties in :root:**
- ✅ No dynamic injection — all values are static hex codes
- ✅ Properties scoped properly in `:root` and `.dark` classes
- ✅ No XSS vectors (CSS properties are not user-input)

**Backward Compatibility:**
- ✅ Existing pages using old Brique & Béton classes will not break (graceful degradation via shadcn-vue defaults)
- ✅ No breaking changes to component APIs
