**Typography Audit:**
- All headings use Figtree (600, 700 weight)
- All body/labels use Noto Sans (400, 500 weight)
- Verify no hardcoded font-family overrides; rely on @apply font-heading / font-sans from tailwind.css

**Icon Check:**
- All icons are SVG (check ✓, arrow →, etc.)
- No emojis anywhere
- Icons have cursor-pointer if clickable (buttons already have this via .inline-flex)

**Spacing Audit:**
- Hero section padding: --space-3xl (64px) on desktop, --space-lg (24px) on mobile (already uses clamp)
- Process section cards: --space-md (16px) gaps
- Ensure no hardcoded px values that break spacing system

**Focus & Accessibility:**
- All buttons have focus-visible:outline ring
- Links have focus states visible (outline or text-underline)
- form inputs (if any) have focus:border-primary focus:ring-primary/20

**Responsive Test Points:**
- 375px mobile: stacked layout, large touch targets (48px+ height for buttons)
- 768px tablet: 2-col layouts where applicable
- 1024px desktop: full grid layout
- No horizontal scroll at any breakpoint

**Avoid Anti-Patterns:**
- ❌ No emoji icons (only SVG)
- ❌ No missing cursor:pointer on clickable elements
- ❌ No layout-shifting hovers (scale transforms forbidden)
- ❌ No low contrast (<4.5:1)
- ❌ No instant state changes (all use 200-300ms transitions)
- ❌ No invisible focus states
