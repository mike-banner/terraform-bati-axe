## Global Design Rules

### Color Palette

| Role | Hex | CSS Variable | Usage |
|------|-----|--------------|-------|
| Primary | `#0891B2` | `--color-primary` | Links, focus states, secondary CTAs |
| Secondary | `#22D3EE` | `--color-secondary` | Hover states, accents |
| CTA/Accent | `#22C55E` | `--color-cta` | Primary buttons, main CTAs |
| Background | `#ECFEFF` | `--color-background` | Page background, light surfaces |
| Text | `#164E63` | `--color-text` | Primary text content |

**Mood:** Fresh cyan + clean green
**Accessibility:** All text maintains 4.5:1 minimum contrast ratio (WCAG AA)

### Typography

- **Heading Font:** Figtree (300, 400, 500, 600, 700)
- **Body Font:** Noto Sans (300, 400, 500, 700)
- **Google Fonts Link:** [Figtree + Noto Sans](https://fonts.google.com/share?selection.family=Figtree:wght@300;400;500;600;700|Noto+Sans:wght@300;400;500;700)

**CSS Import:**
```css
@import url('https://fonts.googleapis.com/css2?family=Figtree:wght@300;400;500;600;700&family=Noto+Sans:wght@300;400;500;700&display=swap');
```

**Semantic Usage:**
- Headings: Figtree 600/700 (bold, professional)
- Subheadings: Figtree 500 (emphasis, hierarchy)
- Body text: Noto Sans 400 (readability, 16px base)
- Labels/UI: Noto Sans 500 (form labels, badges, small text)
- Emphasis: Noto Sans 600 (strong text, error messages)

### Spacing System

| Token | Value | Usage |
|-------|-------|-------|
| `--space-xs` | `4px` / `0.25rem` | Tight gaps, icon spacing |
| `--space-sm` | `8px` / `0.5rem` | Icon gaps, inline spacing |
| `--space-md` | `16px` / `1rem` | Standard padding, card content |
| `--space-lg` | `24px` / `1.5rem` | Section padding, moderate gaps |
| `--space-xl` | `32px` / `2rem` | Large gaps, component spacing |
| `--space-2xl` | `48px` / `3rem` | Section margins, vertical rhythm |
| `--space-3xl` | `64px` / `4rem` | Hero padding, major sections |

**Responsive Padding:**
- Mobile: Use `--space-lg` for section padding (24px)
- Tablet: Escalate to `--space-2xl` (48px)
- Desktop: Full `--space-3xl` (64px) for hero sections

### Shadow System

| Level | Value | Usage |
|-------|-------|-------|
| `--shadow-sm` | `0 1px 2px rgba(0,0,0,0.05)` | Subtle lift, card underlines |
| `--shadow-md` | `0 4px 6px rgba(0,0,0,0.1)` | Cards, buttons, standard depth |
| `--shadow-lg` | `0 10px 15px rgba(0,0,0,0.1)` | Modals, dropdowns, elevated panels |
| `--shadow-xl` | `0 20px 25px rgba(0,0,0,0.15)` | Hero images, featured cards, max depth |

**Motion:** All shadows paired with 200-300ms ease transitions

## Component Specifications

### Buttons

**Primary Button**
```css
.btn-primary {
  background: #22C55E;
  color: white;
  padding: 12px 24px;
  border-radius: 8px;
  font-weight: 600;
  font-family: 'Noto Sans', sans-serif;
  transition: all 200ms ease;
  cursor: pointer;
}

.btn-primary:hover {
  opacity: 0.9;
  transform: translateY(-1px);
  box-shadow: var(--shadow-md);
}

.btn-primary:focus {
  outline: 2px solid #0891B2;
  outline-offset: 2px;
}
```

**Secondary Button**
```css
.btn-secondary {
  background: transparent;
  color: #0891B2;
  border: 2px solid #0891B2;
  padding: 12px 24px;
  border-radius: 8px;
  font-weight: 600;
  font-family: 'Noto Sans', sans-serif;
  transition: all 200ms ease;
  cursor: pointer;
}

.btn-secondary:hover {
  background: #ECFEFF;
  transform: translateY(-1px);
}

.btn-secondary:focus {
  outline: 2px solid #0891B2;
  outline-offset: 2px;
}
```

**Button Sizing:**
- Small: 10px 16px, font-size 14px
- Medium: 12px 24px, font-size 16px (default)
- Large: 14px 32px, font-size 18px

### Cards

```css
.card {
  background: #ECFEFF;
  border-radius: 12px;
  padding: 24px;
  box-shadow: var(--shadow-md);
  transition: all 200ms ease;
  cursor: pointer;
  position: relative;
}

.card:hover {
  box-shadow: var(--shadow-lg);
  transform: translateY(-2px);
}

.card:focus-within {
  outline: 2px solid #0891B2;
  outline-offset: 0;
}
```

**Card Variants:**
- Default: light cyan background (#ECFEFF)
- Featured: shadow-lg + 4px border top (color: --color-primary)
- Minimal: no shadow, border only (1px solid #E2E8F0)

### Form Inputs

```css
.input {
  padding: 12px 16px;
  border: 1px solid #E2E8F0;
  border-radius: 8px;
  font-size: 16px;
  font-family: 'Noto Sans', sans-serif;
  transition: border-color 200ms ease;
  background: white;
}

.input:focus {
  border-color: #0891B2;
  outline: none;
  box-shadow: 0 0 0 3px #0891B220;
}

.input:disabled {
  background: #F1F5F9;
  color: #94A3B8;
  cursor: not-allowed;
}
```

**Input States:**
- Default: border #E2E8F0
- Hover: border #CBD5E1
- Focus: border #0891B2, ring #0891B220
- Error: border #DC2626, ring #DC262620
- Disabled: background #F1F5F9, text #94A3B8

  inset: 0;

## Style Guidelines

**Style:** Vibrant & Block-based

**Keywords:** Bold, energetic, playful, block layout, geometric shapes, high color contrast, duotone, modern

**Best For:** Marketplace, directory, P2P platforms, professional networks

**Key Effects:**
- Large sections (48px+ gaps)
- Smooth transitions (200-300ms)
- Hover states: color shift + subtle lift
- High contrast: cyan + green on white/light cyan
- Large type: 28-48px for headings

---

## Anti-Patterns (Do NOT Use)

### Forbidden Patterns
- ❌ **Emojis as icons** — Use SVG icons (Heroicons, Lucide, Simple Icons)
- ❌ **Missing cursor:pointer** — All clickable elements must have `cursor: pointer`
- ❌ **Layout-shifting hovers** — Avoid scale transforms that shift layout (use shadow/opacity instead)
- ❌ **Low contrast text** — Maintain 4.5:1 minimum contrast ratio
- ❌ **Instant state changes** — Always use transitions (150-300ms)
- ❌ **Invisible focus states** — Focus states must be visible for keyboard navigation
- ❌ **Low trust signals** — Use clear CTAs, trust badges, verified indicators
- ❌ **Confusing layout** — Clear hierarchy, logical flow, no cognitive overload

## Pre-Delivery Checklist

Before delivering UI code, verify:

- [ ] No emojis used as icons (use SVG from Heroicons/Lucide only)
- [ ] All icons from consistent icon set
- [ ] `cursor: pointer` on all clickable elements
- [ ] Hover states with smooth transitions (150-300ms)
- [ ] Light mode: text contrast 4.5:1 minimum (WCAG AA)
- [ ] Focus states visible for keyboard navigation (outline or ring)
- [ ] `prefers-reduced-motion` respected (no animation when disabled)
- [ ] Responsive tested at: 375px, 768px, 1024px, 1440px
- [ ] No content hidden behind fixed navbars (top margin applied)
- [ ] No horizontal scroll on mobile
- [ ] Color palette matches MASTER.md exactly
- [ ] Typography uses Figtree + Noto Sans only
- [ ] Spacing follows spacing system (`--space-*` tokens)
- [ ] Buttons have visible focus states
- [ ] Forms fully keyboard accessible
- [ ] Images have alt text
- [ ] No console errors or warnings
