# UI Design Contract

## Phase: 04.7-UI-Refonte

## 1. Visual Direction
**Theme:** Exaggerated Minimalism & Bento Grids (Awwwards/10x style)
**Core Emotion:** Premium, Industriel, Haute Performance
**Key Reference:** 10x Design principles + 21st.dev micro-interactions

## 2. Typography System
**Primary Font (Headings):** Satoshi / DM Sans
- Weight: 900 (Black) for main titles, 700 for subtitles.
- Tracking: Tightly kerned (`tracking-tighter` / `-0.05em`).
- Sizing: Massive scale (`text-6xl` to `text-8xl` on Desktop, clamp function).

**Secondary Font (Body):** General Sans / DM Sans
- Weight: 400 (Regular) and 500 (Medium).
- Contrast: Muted colors (`text-slate-500`) against dark headings.

## 3. Color Palette
**Backgrounds:**
- Base: `bg-slate-50` (`#F8FAFC`)
- Surfaces: `bg-white` with subtle borders (`border-slate-200`)

**Brand Accents:**
- Primary Structure: Industrial Grey (`#64748B` / `slate-500`)
- High-Conversion CTA: Safety Orange (`#F97316` / `orange-500`)

**Text:**
- Headings: `text-slate-900`
- Body: `text-slate-500`

## 4. Layout & Spacing
**Grid System:** 12-column CSS Grid.
**Bento Cards:** Asymmetrical layouts. Large paddings inside cards (`p-8` to `p-10`).
**Border Radius:** Soft Apple-style curves (`rounded-3xl` for main cards, `rounded-full` for buttons).

## 5. Interaction Patterns (The 21st.dev Touch)
- **Hover States:** Smooth Y-axis translations (`hover:-translate-y-1`) with dynamic drop shadows on Bento cards.
- **Micro-interactions:** Buttons scale up slightly (`hover:scale-105`) with a soft colored shadow (`shadow-safety/20`).
- **Animations:** Staggered reveal for grid items (SlideUp + Fade in).

  safety: '#F97316',
