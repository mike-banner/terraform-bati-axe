<domain>
## Phase Boundary

Créer un levier de réassurance visuel (galerie de réalisations) pour les pros, rafraîchir drastiquement l'UI du profil public en mode immersif (mobile-first), corriger les bugs d'upload R2, et ajouter de la preuve sociale sur la landing page sans dégrader les performances SEO.

</domain>

<decisions>
## Implementation Decisions

### A. Debug Upload Logo
- Identifier et corriger le bug de soumission du logo pro.
- Points d'audit : RLS Supabase sur le bucket R2, configuration CSP (`nuxt.config.ts`), tokens Cloudflare.

### B. Modèle de Données (Base de Données)
- Création de la table `completed_projects` :
  - `id` (UUID)
  - `professional_id` (FK vers `professionals`)
  - `title` (Texte, ex: "Rénovation de toiture")
  - `description` (Texte)
  - `city` (Texte, ex: "Carrières-sous-Poissy")
  - `image_urls` (Tableau de textes, URLs Cloudflare R2)
  - `is_showcased` (Booléen, par défaut `false`, éditable par les admins)
  - `created_at` (Timestamp)
- Création de la table `likes` (pour la preuve sociale) :
  - `id`, `project_id` (FK), `user_ip` ou mécanisme anti-spam basique.

### C. Backoffice (Espace Pro & Admin)
- `/espace/profil` : Bouton "Ajouter une réalisation" ouvrant un formulaire (modal ou page dédiée).
- Upload R2 asynchrone multi-fichiers pour les photos de chantiers.
- Dashboard Admin : Bouton ou toggle rapide pour marquer un projet `is_showcased = true`.

### D. Refonte Profil Public (UI/UX)
- Route : `/pro/[dept]/[slug]`
- Layout : **Pleine page immersive** (suppression de la navbar générique standard de l'application).
- Navigation : Fil d'Ariane (Breadcrumb dynamique) ou boutons flottants minimalistes pour retourner à l'accueil.
- Responsive : Approche **Mobile First** absolue.
- Contenu : Intégration esthétique (Bento Grid) des données existantes (Badges, SIRET, bio) et de la nouvelle galerie de "Réalisations" (avec compteurs de likes).

### E. Landing Page (Accueil public)
- Section "Chantiers Réalisés".
- Carousel fluide SEO-friendly (HTML SSR) ou Grille.
- Affiche uniquement les `completed_projects` où `is_showcased = true`.
- Affiche les compteurs de likes. Bouton like interactif (Optimistic UI).

### the agent's Discretion
- Mécanisme exact anti-spam pour les likes (LocalStorage vs IP vs Supabase Edge Functions).
- Design exact du formulaire d'ajout de réalisation dans l'espace pro.
- Librairie ou composant UI spécifique pour le Carousel fluide SSR.

</decisions>
