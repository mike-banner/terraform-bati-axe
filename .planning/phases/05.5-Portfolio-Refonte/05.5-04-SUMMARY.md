**Endpoints Nitro pour la preuve sociale : liste publique des réalisations showcased avec compteur de likes, réalisations sur le profil pro public, like anti-spam par IP hashée, et toggle admin `is_showcased` audité.**

## Accomplishments
- Landing peut désormais fetcher `/api/v1/completed-projects` pour la galerie showcased (12 max, likes(count), nom d'entreprise)
- Profil public pro (`/api/v1/pro/profile/[slug]`) expose la clé `realisations` sans casser l'invariant `is_verified: false` (pas de 404 sur profil existant non vérifié)
- Un visiteur peut liker une réalisation une seule fois par IP, sans jamais stocker l'IP brute (SHA-256 opaque)
- Un admin peut activer/désactiver `is_showcased` sur une réalisation, tracé dans `audit_logs`
